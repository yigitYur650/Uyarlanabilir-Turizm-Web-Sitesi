Proje Özellikleri
Bungalov Listeleme: Tesislerin özelliklerine ve fotoğraflarına göre detaylı listelenmesi.

Admin Paneli: Bungalov verilerinin (ekleme, silme, güncelleme) merkezi bir panelden yönetimi.

Talep Yönetimi: Kullanıcılardan gelen rezervasyon ve bilgi taleplerinin takibi.

Bölgesel Rehber: Sapanca ve çevresi için hazırlanan özel gezi ve aktivite rehberi.

Dinamik Atmosfer Modu: Kullanıcı deneyimini artıran görsel mod geçişleri.

Responsive Tasarım: Mobil, tablet ve masaüstü cihazlarla tam uyumlu arayüz.

Teknik Detaylar
Frontend (İstemci)
Framework: Next.js (App Router yapısı ile).

Dil: TypeScript.

Stil: Tailwind CSS ve Lucide React ikonları.

Bileşen Yapısı: Yeniden kullanılabilir UI bileşenleri.

Backend (Sunucu ve Veritabanı)
Veritabanı: Supabase (PostgreSQL).

Kimlik Doğrulama: Supabase Auth.

Dosya Saklama: Tesis fotoğrafları için Supabase Storage.

Klasör Yapısı
app/: Sayfa yönlendirmeleri ve ana uygulama mantığı (Admin ve kullanıcı sayfaları).

components/: Arayüzü oluşturan modüler bileşenler (Hero, galeri, iletişim vb.).

lib/: Supabase bağlantısı ve yardımcı fonksiyonlar.

supabase/: Veritabanı şemaları ve migrasyon dosyaları.
